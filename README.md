# Проект homework
## Описание
Проект homework - это учебный проект для приобретения навыков работы Python-разработчика.
Конкретно ведётся разработка виджета банковских операций клиента.
### В проекте реализованы модули generators.py, masks.py, processing.py, widget.py, decorators.py, utils.py, external_api.py.
В каждом модуле реализованы функции соответственно:
1. masks.py
* get_mask_card_number - Функция возвращает маску номера карты, 
видны первые 6 и последние 4 цифры
* get_mask_account - Функция возвращает маску номера счета
в которой видны последние 4 цифры
2. widget.py
* mask_account_card - Функция принимает номер карты или счета
и возвращает в замаскированном виде
* get_date - Функция возвращает дату в формате ДД.ММ.ГГГГ
3. processing.py
* filter_by_state - Функция возвращает отсортированный список по значению 'EXECUTED',
    или список со значением переданным в state_value
* sort_by_date - Функция возвращает отсортированный по дате список(по умолчанию-по убыванию,
    или по возрастанию, если в параметр со значением по умолчанию введено False)
4. generators.py
* filter_by_currency - Функция-генератор принимает список словарей транзакций и возвращает
    итератор, который поочередно выдает транзакции по заданной валюте
* transaction_descriptions - Функция-генератор возвращает описание каждой операции
из списка словарей с транзакциями
* card_number_generator - Функция генерирует номера карт от 0000 0000 0000 0000 до 9999 9999 9999 9999
    по заданному диапазону
5. decorators.py
* log - Декоратор считывает входные данные с декорируемой функции, а так же результат её работы.
    Если не указан необязательный аргумент, результат выводится в консоль.
    Если указан необязательный аргумент, результат записывается в файл, который указан в аргументе.
6. utils.py
* get_list_transactions - Функция принимает файл формата json, и возвращает список транзакций.
* get_amount_rub - Функция принимает данные транзакции и возвращает сумму транзакции в рублях.
    Если транзакция в USD или EUR, происходит конвертация в RUB по текущему курсу.
7. external_api.py
* convert_to_rub - Функция конвертирует сумму в RUB по текущему курсу.

## Тестирование
1. Для тестирования использовать Pytest. Установка через Poetry:
```
poetry add --group dev pytest
```
2. В проекте реализован пакет tests с тестами для каждой функции из соответствующего
модуля в пакете src
3. Тестирование модуля выполнять при помощи команды
```
pytest tests\<имя модуля из пакета tests>
```
4. Информация о покрытии кода тестами в [документе](htmlcov\index.html)
## Логирование
В проекте ведётся логирование функций из модулей external_api.py, utils.py, masks.py.
Результаты логирования сохраняются в папке logs отдельно для каждого модуля
